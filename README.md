# CustomViewPractice
Custom View를 연습해보자
<br>

<h2>프로젝트 개요</h2>
- 반복되는 기능을 하는 위젯/레이아웃 은 Custom View로 따로 만들어서 반복해서 사용하도록 만들 수 있다는 이야기를 전해들었다.<br>
- 반복되서 사용될 법한 커스텀 뷰들의 예제를 만들어가면서 커스텀뷰에 대해 공부해보도록 하자.<br>
-- 예제1) N초 안에 여러번 클릭을 방지하는 버튼<br>
-- 예제2) N초마다 자동으로 넘어가는 Image Banner 레이아웃 만들기<br>
